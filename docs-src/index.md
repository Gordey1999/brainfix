# Документация Brainfix

Добро пожаловать на сайт документации **Brainfix**!

Brainfix — это Си-подобный строго типизированный язык высокого уровня, созданный для трансляции в оптимизированный и компактный код Brainfuck. Здесь вы найдёте исчерпывающие руководства по синтаксису языка, практические примеры использования и инструкции по работе со встроенной веб-средой разработки.


## Разделы руководства

* **[Полное руководство по синтаксису](./language.html)** — Детальное описание базовых возможностей и правил языка.
* **[Интерфейс и возможности Brainfix Studio](./studio.htlm)** — Подробный разбор официальной веб среды разработки.
* **[Практические примеры программ](./examples.html)** — Каталог готовых алгоритмов на Brainfix с подробным построчным разбором логики.

## Полезные ресурсы

* **[Brainfix Studio (Web-IDE)](../)** — официальная онлайн-среда разработки. Пишите, компилируйте и отлаживайте свои программы прямо в браузере.
* **[Исходный код на GitHub](https://github.com/Gordey1999/brainfix)** — публичный репозиторий проекта. Вы можете изучить устройство компилятора, сообщить об обнаруженном баге или предложить своё улучшение.
